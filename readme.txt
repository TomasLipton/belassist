== Description ==

1. Распакуйте архив и загрузите “assist-for-woocommerce” в папку ваш-домен/wp-content/plugins или установите плагин через панель администрирования: <ваш сайт>/wp-admin/plugin-install.php?tab=upload

2. Активируйте плагин в панели управления, в разденле Плагины > Установленные.

3. Зайдите в настройки WooCommerce (WooCommerce > Настройки), на странице "Оплата" выбираем Assist и прописываем:
Merchant_ID, секретное слово, адрес сервера - их вы получите после подключения к системе Assist (Заполнить заявку на подключение можно на странице http://www.assist.ru/shop_register.htm)

В личном кабинете Assist, в настроках мерчанта, в разделе "Настройка отправки результатов платежей" настраиваем:
- Отправка результатов оплат - да
- Отправка результатов отмен - да
- отправка подтверждений - да
- отправлять только успешные - нет
- URL для отправки результатов: http://<домен вашего сервера>/?wc-api=wc_assist&assist=result
- Тип протокола - POST
- Тип подписи - MD5
- Секретное слово: <ваше секретное слово>
